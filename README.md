# E-Commerce Sales Analysis Dashboard

## Project Overview

This project analyzes e-commerce sales data to uncover insights related to sales performance, profitability, customer behavior, product performance, and regional trends. The dashboard was built using Power BI to help stakeholders make data-driven decisions.

## Tools Used

* Power BI
* SQL
* Microsoft Excel

## Key Performance Indicators (KPIs)

* Total Sales
* Total Profit
* Total Orders
* Profit Margin %

## Key Insights

* Technology category generated the highest sales.
* Consumer segment contributed the largest share of revenue.
* A small group of customers contributed significantly to total sales.
* Regional analysis helped identify high-performing states and profit-driving regions.
